# Simple Node React Starter

### NPM Scripts

###### Build on Dev:
```
npm run build:dev
```
###### Build on Prod:
```
npm run build
```
###### Run Node on Prod:
```
npm run dev
```
###### Run Node on Prod:
```
npm run start
```
