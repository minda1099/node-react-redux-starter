# Simple Node + React Redux Starter

This is currently work in progress.  End goal is to create a starter pack with all the tooling
to quickly build a React Redux web app.

### NPM Scripts

###### Build on Dev:
```
npm run build:dev
```
###### Build on Prod:
```
npm run build
```
###### Run Node on Prod:
```
npm run dev
```
###### Run Node on Prod:
```
npm run start
```
