### NPM Scripts

###### Test Server:
```
npm run test:server
```
###### Test App:
```
npm run test:app
```
###### JS Linter:
```
npm run lint
```
###### Build on Dev:
```
npm run build:dev
```
###### Build on Prod:
```
npm run build
```
###### Run Node on Prod:
```
npm run dev
```
###### Run Node on Prod:
```
npm run start
```
